## Prerequisites

You'll need [Bun](https://bun.sh) installed (or Node 20+ with npm). That's it.

## Configure your site

All site-wide settings live in a single file. Open `src/site.config.ts` and update the identity block:

```ts
export const SITE = {
  website: "https://yourdomain.com",
  title: "Your Blog Name",
  author: "Your Name",
  description: "One-line description of your blog.",
  // ...
};
```

## Customise navigation

The nav links are an array in `site.config.ts`. Each item needs a `label` and an `href`:

```ts
nav: [
  { label: "Blog", href: "/posts/" },
  { label: "Tags", href: "/tags/" },
  { label: "About", href: "/about/" },
],
```

Add, remove, or rename as needed. The About page lives at `src/data/pages/about.md` — edit it directly.

## Light and dark mode

The theme toggle is included by default. You can disable it by setting `lightAndDarkMode: false` in `site.config.ts`. The chosen theme is stored in `localStorage` and persists across visits.
